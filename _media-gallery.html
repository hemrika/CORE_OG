.media-gallery

This is our gallery "lightbox" powered by Venobox. A light, responsive plugin used to display media inside a detailed view. You need to specify a thumbnail image and a link to a detailed image or video URL. Note that all the thumbnails MUST be cropped the at the same aspect ratio and and should be the same pixel size to avoid any distortion

What makes this lightbox different is that is allows for taller sized items to be viewed by scrolling instead of shrinking the content to match the viewport. This is especially great for previewing web site templates or items that are vertically longer than others. Click on the first item for an example of this detailed view.

Media Gallery

Our gallery is based on the "media-posts" module. By default we use "flexbox" to achieve the layout with emphasis on the newest post and offers an alternative grid look. If a more traditional layout is preferred consider using the "media-gallery-compact" panel as an option or modify the CSS.

Media Gallery Compact

This gallery is available and recommended if more than 16 items are needed to display. These posts can also be sorted with Kentico's pager functionality that could help optimize the amount of items showing per page as well as the amount of pages total.
