Modals

Our modal functionality is powered by remodal.js, the:

"Responsive, lightweight, fast, synchronized with CSS animations, fully customizable modal window plugin with declarative configuration and hash tracking".

A modal is initiated by linking to the specified modal's unique ID. You can place multiple content types inside a modal and can initiate them with multiple links if desired. In this case we are calling #modal-one.

You can also have multiple modals on one page as show in the examples demo below.

Each modal MUST have a unique ID to display the related content and function independently from each-other. Use cases include: media or content details, contact forms, login/signup/user prompts.
